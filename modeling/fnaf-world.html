FNAF World: Fan-Assets
Commissions & Personal

â“˜   This page holds a selection of modeled content based off of the Five Nights at Freddy's spin-off game, FNAF World. It is well known for the FNAF community to create non-profit/non-commercial fan-assets for fan-animations, but any assets for this particular spin-off were almost nonexistent. I took it upon myself to create these reusable assets for others for use in Blender and Source Filmmaker. These assets were released into packs, in 2022 and 2023.

All FNAF World assets are made to be as accurate to the original as possible. Certain assets shown here are also a part of fangames that mimic the same style. (Can you tell which ones they are? They're the icicle, left-most house, jungle tree, and flowers/clovers!)

The last image shows a test render I created with all of the assets I created to see how well the style fits in a 3D world. The original game is a top-down 2D platform, so being able to translate these models well enough for use in 3D space was key.
